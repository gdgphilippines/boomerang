Add HTML template to Gmail

  1. Compose a new message
  2. Remove all text inside your email including the signature
  3. Add a sample text such as '****'
  4. Highlight the sample text
  5. Right click on the sample text
  6. Select Inspect
  7. Right click on the parent element of the sample text
  8. Select Edit as HTML
  9. Paste the code below inside the element
  10. Press Ctrl+Enter to exit and save your code
  11. Edit your content through Gmail
